# Introduction
jgl (Jordan GL) library for common graphics stuff.

# Features
 * Support for ModelViewProjection matrices
 * Materials
 * Material JSON reader
 * Ray/HitRecord
 * SceneNode/SceneGraph
 * Primitives
     - Sphere
     - Triangle
     - Plane
     - TriMesh

# Get
go get http://github.com/Stymphalian/go.gl/jgl
